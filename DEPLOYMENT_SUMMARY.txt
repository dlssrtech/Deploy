🚀 QUICK START (GitHub Pages - FREE)
═══════════════════════════════════════════════════════════════════════

Step 1: Create GitHub Repository
─────────────────────────────────
1. Go to https://github.com/new
2. Name: topdesign-website
3. Make it PUBLIC
4. Click "Create repository"

Step 2: Upload Files
─────────────────────────────────
Option A: Using Git (Recommended)
   cd topdesign-github
   bash setup.sh

Option B: Manual Upload
   - Download topdesign-github.zip
   - Extract and upload all files to GitHub

Step 3: Enable GitHub Pages
─────────────────────────────────
1. Repository → Settings → Pages
2. Source: "Deploy from a branch"
3. Branch: main / (root)
4. Click Save

Step 4: Your Site is Live!
─────────────────────────────────
URL: https://YOUR_USERNAME.github.io/topdesign-website

Step 5: Custom Domain (www.topdesign.co.in)
─────────────────────────────────
1. In GitHub Pages settings, add: www.topdesign.co.in
2. At your domain registrar, add CNAME:
   Name: www
   Value: YOUR_USERNAME.github.io
3. Enable "Enforce HTTPS"
